This project creates a chat server in terminal using Node.js and sockets, a link to the docs for Node can be found below.

To login and use the chat app you use the
nc localhost 3000 command.

You can change your autogenerated unique id using the @nickname <new name> command.

You are able to direct message others in the channel by using the @dm <user id> command.

To use the app you must install the UUID package from npm. The docs are linked below.

https://nodejs.org/dist/latest-v6.x/docs/api/net.html#net_net_createserver_options_connectionlistener

https://www.npmjs.com/package/uuid
